# json

## What is this?

This is my playground for learning about lexing, parsing, and json in general. This project intentionally has no dependencies

### Tests

I wrote all of the tests using bun:test. You can learn more about how to install bun [here](https://bun.sh/)

By default, all tests will be executed. If you are after a specific file, you can specify via:

```bash
bun test lexer
```

You can also utilize `--random` or `--rerun-each` for some added confidence
